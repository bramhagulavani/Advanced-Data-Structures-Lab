# Advanced DSA Coding

A comprehensive collection of Advanced Data Structures and Algorithms implementations in C.

## Overview

This repository contains implementations of various advanced data structures and algorithms that are commonly used in competitive programming and software interviews.

## Contents

### Tree Data Structures
- **AVL.c** - AVL Tree (Self-balancing binary search tree)
- **BTree.c** - B-Tree implementation
- **B+Tree.c** - B+ Tree implementation
- **RBT.c** & **RedBlack.c** - Red-Black Tree implementations
- **inorderTBT.c** - Inorder Traversal of Threaded Binary Trees

### Graph Algorithms
- **Dijkstra's Algorithm.c** & **DijkstraAlgorithm.c** - Dijkstra's shortest path algorithm
- **RandomisedDataStructure.c** - Randomized data structure implementations

### Priority Queues & Heaps
- **priorityqueue.c** - Priority Queue implementation
- **priorityHospital.c** - Priority Queue application (Hospital/Scheduling system)
- **heapsort.c** - Heap Sort algorithm

### String Processing
- **prefix.c** - Prefix-related operations
- **suffix.c** - Suffix-related operations (e.g., Suffix Array, Suffix Tree)

## Getting Started

### Prerequisites
- GCC compiler or any C compiler
- Basic knowledge of data structures and algorithms

### Compilation

To compile a specific program:
```bash
gcc -o program_name program_name.c
```

For example:
```bash
gcc -o AVL AVL.c
./AVL
```

## Usage

Each file contains a complete implementation with a main function that demonstrates the data structure or algorithm. Simply compile and run the executable:

```bash
gcc -o output_file source_file.c
./output_file
```

## File Organization

- Each `.c` file is a standalone program
- Each file contains the complete implementation and driver code
- Executables (.exe files) can be run directly on Windows systems

## Topics Covered

- **Self-Balancing Trees**: AVL Trees, Red-Black Trees, B-Trees
- **Advanced Tree Structures**: Threaded Binary Trees, B+ Trees
- **Graph Algorithms**: Dijkstra's shortest path algorithm
- **Heap-based Algorithms**: Heap Sort, Priority Queues
- **Randomized Data Structures**: Randomized algorithms and data structures
- **String Algorithms**: Prefix and Suffix based algorithms

## Notes

- All implementations use standard C libraries
- Each program includes a main function with test cases
- Compiled executables are provided for convenience
